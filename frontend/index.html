Project Details

About the Frontend Project: Stock Market Visualization Dashboard This project is a full-stack Stock Sentiment Analysis application designed to give traders a quick and powerful insight into market mood. The backend is built with Python using the high-performance FastAPI framework, which powers an API that fetches recent, relevant news articles for any given stock ticker from the News API. Once the news data is retrieved, the core of the analysis is performed by TextBlob, a Natural Language Processing (NLP) library that reads each article's headline and description to calculate a sentiment score. This score is then averaged and classified as Positive, Negative, or Neutral. The entire user experience is handled by a clean and interactive frontend built with HTML, CSS, and JavaScript, which sends the user's request to the backend and dynamically displays the sentiment results without a page reload. By combining real-time news analysis with a visual TradingView chart, the tool offers a unique edge, helping investors understand the "why" behind price movements and make more informed, data-driven decisions.

This web application serves as an intelligent tool for analyzing market sentiment surrounding publicly traded stocks. Rather than predicting exact stock prices, it leverages the power of Natural Language Processing (NLP) to gauge the emotional tone of recent news coverage. The core principle is that media sentiment is a significant, often leading, indicator of stock market movements. The user is greeted with a clean interface featuring a real-time TradingView chart for visual analysis. By simply entering a stock ticker symbol (like 'GOOGL' for Alphabet or 'TSLA' for Tesla), the application's backend is triggered to perform a comprehensive sentiment analysis, presenting the user with a final score classified as Positive, Negative, or Neutral. Technology Stack & Architecture ⚙️ The project is built on a modern client-server architecture, ensuring a responsive user experience and efficient data processing. Backend (Server-Side Logic) Python: The foundational language chosen for its extensive data science and web development libraries. FastAPI: A high-performance Python framework used to build the API. It handles incoming requests from the user's browser, processes the logic, and sends back the results quickly and efficiently. News API: This external, third-party service is integrated to fetch the latest, most relevant news articles from thousands of global sources for any given stock symbol. TextBlob: A powerful Python library for NLP. It is the core engine of the analysis, responsible for reading the headlines and descriptions of the news articles and calculating a "polarity" score—a measure of how positive or negative the text is. Frontend (User Interface) HTML5: Provides the essential structure and content of the webpage, from the input form to the results display area. CSS3: Handles all visual styling, layout, and presentation, ensuring the application is intuitive and aesthetically pleasing. JavaScript: Brings the page to life by enabling interactivity. It captures the user's input, communicates with the backend FastAPI server using the Fetch API, and dynamically updates the page with the sentiment analysis results without requiring a page refresh.
